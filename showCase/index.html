使用可能な色一覧

色定義は使用可能な色が決まっています。以下の色を使用しましょう。

変数名: $clr-black

文字色: #000000

変数名: $clr-background

文字色: #EFEDE9

変数名: $clr-purple

文字色: #9f5edf

変数名: $clr-menu

文字色: #868282

変数名: $clr-white

文字色: #ffffff

文字定義について

文字は以下のサイズ、色で定義されています。それ以外の文字サイズ、色は使用しないようにしましょう。

文字サイズは、タイトル24px、本文 18px、行間 1.5pxで基本的には定義してください。

  • scssに記述する場合(@includeを用いる事によって、以下の形式を使用することができます)

    ※scssファイルに記述する場合、エディタの変換機能を用いて、それぞれのcssファイルに変換する必要があります。

    1、セクションタイトル

    サンプル

    scss: @include text-section-title

     css: 
                                                
.text-section-title{
  padding-bottom: 10px;
  font-weight: bold;
  font-size: 24px;
  border-bottom: #000000 solid 2px;
  font-family: YuGothic Arial, sans-serif;
}

    この文字はセクションごとのタイトル（下線入り）で使用します

    2、本文

    サンプル

    scss: @include text-body

     css: 
                                                
.text-body{
  font-size: 18px;
  font-family: YuGothic Arial, sans-serif;
}

    この文字は本文で使用します。
